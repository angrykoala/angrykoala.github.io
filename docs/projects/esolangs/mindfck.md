# Mindfck

Mindfck is a simple language with a C-style syntax that transpile to [Brainfuck](https://en.wikipedia.org/wiki/Brainfuck)

-   [GitHub](https://github.com/angrykoala/mindfck)

For example:

```
int a
int b
a = 3 + a
a = 33 + a
a = a + 2
int c
a = a + 21
a = a + 2
b = 10 + a
c = a + b
print c
```

## Features

-   Variables
    -   Byte, can be assigned with numbers or char values
    -   Int, unsinged 16-bit ints.
-   Math operations
-   Logical operations
-   If ... Else
-   While loops
-   Print and read bytes
